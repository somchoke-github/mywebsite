TPS 4 Steps

  Step 1. Work Site Control

  Step 2. Continuous Flow

  Step 3. Standardized Work

  Step 4. Pull System


1. Work Site Control คือ การทำให้สถานที่ทำงานอยู่ในสภาพที่ได้รับการควบคุมดูแล

2. Continuous Flow คือการผลิตแบบที่ละชิ้นที่ละชิ้นให้งานไหลอย่างราบรื่น โดยไม่หยุดชะงัก


3. Standardized Work คือมาตรฐานที่ใช้ในการผลิต โดยรวบรวมการเคลื่อนไหวของ คนเป็นหลัก เพื่อให้มีวิธีการผลิตอย่างมีประสิทธิภาพมากที่สุดตามลำดับขั้นโดยไม่มีมุดะ

4. Pull System คือ การผลิตแบบดึง งานจากกระบวนการก่อนหน้าเพื่อมาผลิตในกระบวนการของตัวเองและวางงาน ที่ทำเสร็จไว้ท้ายกระบวนการ
